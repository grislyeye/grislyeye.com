# Grisly Eye Games

Static HTML site for Grisly Eye Games.

Requirements:

- [Node.js](http://nodejs.org/).

To set-up your environment execute:

```bash
npm i
```

To build:

```bash
npm run build
```

To test:

```bash
npm test
```

To run:

```bash
npm start
```

And open <http://localhost:8080> in your browser.

To start a new post:

```bash
npm run post:new
```

Or, a new product:

```bash
npm run product:new
```

Or, a new review:

```bash
npm run review:new
```
